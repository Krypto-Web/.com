# DevHub | Web Utility & Diagnostic Suite

A sleek, dark-themed dashboard for real-time network diagnostics and developer tools.

## 🚀 Key Features
- **Network Intelligence:** Live tracking of Public IP, ISP, and Location.
- **Device Analytics:** Monitor OS, Browser, Screen Resolution, and Battery Status.
- **Utility Toolkit:** - 🔐 Password Generator (Secure 16-char)
  - 📦 JSON Beautifier/Formatter
  - 📝 Text Analyzer (Word/Char Count & Case Converter)
  - 🔗 Base64 Encoder/Decoder
  - ⚖️ PX to REM Converter
  - 🎨 Hex Color Picker

## 📦 Setup
1. Download `index.html`, `style.css`, and `script.js` into one folder.
2. Open `index.html` in any modern browser.

## 💰 Monetization
Includes pre-styled `ad-box` containers for seamless **Google AdSense** or **Ezoic** integration.
